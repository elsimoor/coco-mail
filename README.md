```markdown
# Cocoinbox — Starter Monorepo (Next.js + Node.js + TypeScript + GraphQL)

This repository is a starting scaffold for "Cocoinbox – Super-application de confidentialité" (privacy-first super-app) using:

- Frontend: Next.js + TypeScript
- Backend: Node.js (Express) + TypeScript
- API: GraphQL (Apollo)
- Monorepo layout: `/frontend`, `/backend`, `/shared`

Included:
- Responsive dashboard layout (Layout + Sidebar + Navbar) in the frontend.
- Minimal Apollo Server in the backend exposing a basic GraphQL schema.
- A checklist of inferred features to track progress.

Local development
- Frontend:
  - cd frontend
  - npm install
  - npm run dev
- Backend:
  - cd backend
  - npm install
  - npm run dev
```
